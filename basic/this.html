# this对象

this 对象是在运行时基于函数的执行环境绑定的：在全局函数中， this 等于 window，而当函数被作为某个对象的方法调用时， this 等于那个对象。

# 函数调用、匿名函数

指向全局对象

function test() {
  this.x = 1
  console.log(x)
}
test()
console.log(x)// 1

# 对象方法

指向上一级对象

var name = 'global this'
let o = {
  name: 'object this',
  sayName: function() {
    console.log(this.name)
  }
}
o.sayName() // object this

# call/apply/bind

切换函数执行的上下文环境（context），即 this 绑定的对象。 区别：call、apply绑定立即执行，call参数分别传，apply参数以数组形式传；bind绑定之后并不会立即执行，需要再调用一次

let a = {
  x: 1,
  log: function(y) {
    console.log(this.x, y)
  }
}

let b = {
  x: 11,
  log: function(y) {
    console.log(this.x, y)
  }
}

a.log(2) // 1 2
a.log.call(b, 2) // 11 2
a.log.apply(b, [2]) // 11 2
a.log.bind(b, 2)() // 11 2
