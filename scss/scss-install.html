    # 概况

    因为网络问题，sass-loader和node-sass可能安装失败，这里可以采用淘宝镜像解决一下

    # 安装镜像

    淘宝 NPM 镜像

    npm i -g cnpm --registry=https://registry.npm.taobao.org
    1

    # 项目安装依赖

    在镜像安装成功之后，切换到项目，安装依赖

    cnpm i sass-loader node-sass -D
