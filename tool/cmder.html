    Cmder

    命令行工具 Cmder

    因为系统自带的命令行使用不便，于是想换一款更强的命令行工具，简单记录配置。

    # 下载

    官网上下载 cmder_mini版本一般就足够了，配置强烈的可以下载完整版本。

    # 使用

    解压，然后配置环境变量

    • 打开

    右键我的电脑，选择属性

    • 配置环境变量

    选择高级系统设置 -> 环境变量 -> 系统变量 path -> 新建一个路径，路径是解压的路径 -> 确定

    # 全局使用

    以管理员身份运行 cmd ，执行

    Cmder.exe /REGISTER ALL
    1

    之后就可以在文件夹右键显示
